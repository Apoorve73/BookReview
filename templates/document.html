API DOCUMENTATION

https://vast-mesa-71372.herokuapp.com/api/{isbn}, where substitute the {isbn} with the isbn number to get json data in the following format

{
                    "author": "author of the book",
                    "average_score": "average score",
                    "isbn": "isbn number",
                    "review_count": "reviews left",
                    "title": "title of the book",
                    "year": "year published"
            }

For example, if isbn=0553588451, then https://vast-mesa-71372.herokuapp.com/api/{isbn} = https://vast-mesa-71372.herokuapp.com/api/0553588451 will give the following json object:-

{
                    "author": "Keri Arthur",
                    "average_score": "3.89",
                    "isbn": "0553588451",
                    "review_count": 49397,
                    "title": "Full Moon Rising",
                    "year": 2006
            }
